# Cloud-Native Task Management Backend

Spring Boot REST API for task management built with clean architecture, JPA, H2, Swagger, Docker-ready and AWS deployable.

## Tech Stack
- Java 17
- Spring Boot
- Spring Data JPA
- H2 Database
- Swagger / OpenAPI
- Docker (upcoming)
- AWS (upcoming)

## Features
- Create tasks
- Fetch all tasks
- In-memory DB with JPA auto schema
- API documentation via Swagger

## Run Locally
```bash
mvn clean install
mvn spring-boot:run
